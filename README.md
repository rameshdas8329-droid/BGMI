# BGMI ERP System

This is a web-based ERP system for BGMI players.

## Features
- Login using BGMI UID
- Track match attendance
- View player ranking
- Clean and animated UI

## Tech Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open the website
2. Login using your BGMI UID
3. View attendance and rank
